# Claude Code System Inventory

This repository contains a comprehensive inventory of all applications, tools, automations, and documentation built with Claude Code.

## Purpose

This central registry helps track:
- Applications and their status
- Automation systems and workflows
- Claude skills and capabilities
- Development projects
- Documentation locations
- Configuration files
- LaunchAgents and scheduled tasks

## Files

- **CLAUDE_INVENTORY.md** - The complete system inventory (main document)
- **README.md** - This file

## Usage

This inventory is automatically synced between:
- Local: `/Users/rohitkaul/CLAUDE_INVENTORY.md`
- GitHub: This repository

Claude Code reads this file on startup to understand the current system state.

## Updating

When you create new apps, tools, or automations with Claude Code, update the `CLAUDE_INVENTORY.md` file and changes will be automatically synced to GitHub.
